The AI Generated Dictionary

Every word definition is generated using artificial intelligence

New words are added every day to our database

Data is generated using OpenAI Text Completion

The worlds first dictionary generated using the help of artificial intelligence

The accuracy of this dictionary will not be as useful or trustworthy as each word is entirely generated by a computer program. There is no human that goes through to make sure that everything looks good, whatever the AI spits out is what is saved. This can funny at times with some of the definitions being completely unrelated to the word being defined. We will however delete content that is innapropiate or offensive.

All data stored on each word

  • Word definition
  • Exmaple sentences using the word
  • Synonyms and antonyms of the word
  • The history and use of the word
  • Reference links that help users understand word better

It is very impressive to see what these text completion and image generation models are capable of. Within seconds it can deliver high-quality definitions for any word in the English dictionary (most of the time) along with pictures all from a basic prompt. There are many instances where a word has multiple meanings and the AI is only able to define one of these instead of all of them. This will be fixed when the model gets more advanced with its responses.
